1.Какие профессиональные компетенции формируются в ПМ.02?
В ПМ.02 формируются компетенции, связанные с разработкой программных продуктов, проектированием баз данных, использованием языков программирования (например, Python), работе с системами управления базами данных (MySQL), созданием пользовательских интерфейсов (tkinter), а также навыки работы с системами контроля версий (Git).

2.Почему выбран стек: Python + MySQL + tkinter? Какие альтернативы могли быть?
Этот стек выбран за его простоту, популярность и богатую экосистему. Python - легкий в изучении язык с мощными библиотеками, MySQL - надежная система управления базами данных, tkinter - встроенная библиотека для создания графических интерфейсов. Альтернативы:  Java, C#, JavaScript, PostgreSQL, SQLite, MongoDB, PyQt, wxPython, Kivy

3.Что такое ER-диаграмма? Для чего она нужна?
ER-диаграмма - это графическое представление структуры базы данных, показывающее сущности (таблицы) и связи между ними. Она нужна для проектирования базы данных, определения структуры данных и их связей до начала реализации.

4.Какие типы связей между таблицами существуют? Приведите примеры из своего варианта.
Основные типы связей:  Один к одному(например, одна таблица содержит информацию о пользователе, а другая — его профиле.), один ко многим(например, таблица клиентов и таблица заказов, где один клиент может иметь много заказов.), многие ко многим (например, таблица студентов и курсов, где студент может записаться на много курсов, а курс — иметь много студентов (требуется связующая таблица)

5.Что такое Git? Чем отличается git commit от git push?
Git - система контроля версий, позволяющая отслеживать изменения в коде и работать в команде.  git commit — сохраняет текущие изменения в локальный репозиторий.  git push — отправляет зафиксированные изменения с локального репозитория на удалённый сервер (например, GitHub).

6.Как проверить, что Python установлен и tkinter работает?
python --version
Для проверки tkinter:Создайте файл test_tkinter.py с кодом:  
import tkinter as tk
root = tk.Tk()
root.mainloop()
Запустите его командой:  
python test_tkinter.py

7.Какие меры безопасности нужно соблюдать при работе за компьютером?  
Использовать антивирусное программное обеспечение.  
Регулярно обновлять системы и программы.  
Использовать сложные пароли и двухфакторную аутентификацию.  
Не открывать подозрительные письма и ссылки.  
Делать резервные копии данных.  
Ограничивать физический доступ к компьютеру.

8.Что означает IP-адрес 127.0.0.1?
Это локальный IP-адрес (localhost), который указывает на текущий компьютер. Он используется для обращения к самому себе через сеть.

9.Как добавить файл в репозиторий Git (последовательность команд)?
Последовательность команд:  
git add <имя_файла>
git commit -m "Описание изменений"
git push

Или, чтобы добавить все файлы:  
git add .
git commit -m "Описание изменений"
git push

10.Зачем нужен файл README.md в проекте?
README.md содержит описание проекта: его назначение, инструкции по установке, использованию, структуре проекта и другую важную информацию. Он помогает другим понять, что делает проект, и как с ним работать.
